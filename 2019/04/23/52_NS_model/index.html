NS/NSS model for term structure prediction and other analysis

This repo contains codes for my girl friend’s undergraduate thesis.

She contributed her ideas but could not implement it. I found that there are some NS/NSS model implementation in python/R/matlab. Some models rate and maturity, some like QuantLib allows users model price and maturity. The jupyter notebook and R markdown will make it easier for a naive user to learn to run and use the model.

There are some requirements like Quantlib and fbonds package. Although not listed together, you can find the instruction of how to install them.

Some other analysis including PCA and a 3D+contour plot. You can also find the plot settings in figure_template.py quite useful.

Some codes are not the final version cause I wrote some codes in my girl friend’s mac. All the codes can be run by a simple beginner :-D

By the way, I am wondering about the true quality of most of undergraduate thesis. It seems it only needs me less than three days to understand all the concepts, principles, tools, models to implement all the codes and get all the results for my girl friend’s thesis. However, it seems my 0.5 hour’s work is equal to her week’s effort. It’s quite interesting that a biology major student may have some talent in solving economic questions. Maybe I choose the wrong field :), who knows?
